# Build: Constraint Enforcement Plan

| Red Line | Axis | Mechanism |
|---|---|---|
| 断定的・射幸的表現禁止 | Think | 生成前にVOとJP-scriptを自己レビュー。「必ず」「絶対」「確実」「保証」を含むセルは即修正 |
| 画面列は中文のみ | Do | 生成後、画面列の全セルに日本語文字（ひらがな・カタカナ・漢字+和語）がないか確認。ある場合は中文に変換してから次のステップへ |
| 平假名表記（片假名禁止） | Do | 【平假名】列の全セルにカタカナ（ア-ン）が含まれていないか確認。含まれる場合は平假名に変換 |
| エンドカット旧協会名禁止 | Do | エンドカット行のdisclaimer列に「資産運用業協会」が含まれることをチェック。「投資顧問業協会」が残っていれば差し替え |
| 比較表現に免責必須 | Do | 「業界初」「業界最安」「No.1」「他社比」を含むJP-scriptまたはVO行のdisclaimer列が空でないことを確認 |
| 見出し半角30文字以内 | Do | 各見出しを生成後に文字数カウント（全角=2、半角=1）。超過の場合は短縮してから出力 |
| 説明文半角90文字以内 | Do | 各説明文を生成後に文字数カウント。超過の場合は短縮してから出力 |
| 動画尺30-35秒 | Think | タイムライン設計時に合計秒数を確認。短すぎる場合はシーンを追加、長すぎる場合は圧縮 |

**Do軸の割合**: 7/8 = 87.5%（基準の30%を大幅に超過）
