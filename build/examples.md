# Build: Examples

## Example 1 — 決算シーズン篇（本番実例）
**Input**:
- 渠道: Google/YouTube/Yahoo
- 人群: 米株進階/高頻交易人群
- 背景・故事線: 美股財報季来臨，如何把握這個賺效？在moomoo你可以在財報發布前/中/後使用各種功能抓住投資機會

**Output**:
- 社内コンプラチェック用広告文（2バリエーション）: 見出し≤30半角 + 説明文≤90半角
- 脚本テーブル（8行, 30-33秒）: 画面(中文)/时长/中文script/JP-script/VO/平假名/screenshot/disclaimer
- 飛書クラウドドキュメントとして作成済み

**Failure Case**:
- 画面列に日本語→設計師が読めない
- 片假名でVO表記→正しくは平假名
- エンドカット免責文に旧協会名「日本投資顧問業協会」→2026/4/1更新済の「資産運用業協会」が正しい

## Example 2 — 初心者層向け（円安・米国株）
**Input**:
- 渠道: YouTube
- 人群: 米国株初心者、30-40代
- 背景・故事線: 円安時代、預金だけでは資産が目減り。米国株をmoomooで始めよう

**Output**:
- 広告文 + 脚本テーブル（30-35秒）
- 断定的・射幸的表現なし（「必ず儲かる」等NG）
- 飛書ドキュメント作成

**Failure Case**:
- 「確実に利益が出る」等の断定表現→日証協違反
- 免責文なしで株価数値を表示→コンプライアンス違反

---

## Example 3 — M7・長期投資篇
**Input**:
- 渠道: Google Display
- 人群: 長期投資家、40-50代
- 背景・故事線: マグニフィセント7に注目が集まる中、moomooでアナリスト評価・目標株価を無料確認

**Output**:
- 広告文 + 脚本テーブル
- M7ロゴ使用→第三者ロゴ免責必須
- 目標株価表示→TipRanks免責必須

**Failure Case**:
- M7ロゴ使用時に第三者免責を付与しない
- 「業界初」表現に比較免責なし
